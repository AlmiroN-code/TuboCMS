# SEO метатеги для категорий

## Описание

Добавлена возможность настройки SEO метатегов для каждой категории отдельно.

## Новые поля в Category

- `metaTitle` (VARCHAR 255) - Заголовок страницы для SEO
- `metaDescription` (TEXT) - Описание страницы для SEO
- `metaKeywords` (VARCHAR 500) - Ключевые слова через запятую

## Использование

### В админ-панели

При создании или редактировании категории доступна секция "SEO метатеги" с полями:

1. **Meta Title** - если не указан, используется название категории
   - Рекомендуемая длина: 50-60 символов
   
2. **Meta Description** - если не указано, используется описание категории
   - Рекомендуемая длина: 150-160 символов
   
3. **Meta Keywords** - ключевые слова через запятую
   - Пример: видео, категория, контент

### На фронтенде

На странице категории (`/categories/{slug}`) автоматически используются:

- `<title>` - metaTitle или название категории
- `<meta name="description">` - metaDescription или описание категории
- `<meta name="keywords">` - metaKeywords

## Миграция

Миграция `Version20260114231415` добавляет три новых поля в таблицу `category`:

```sql
ALTER TABLE category 
ADD meta_title VARCHAR(255) DEFAULT NULL, 
ADD meta_description LONGTEXT DEFAULT NULL, 
ADD meta_keywords VARCHAR(500) DEFAULT NULL
```

## Рекомендации

1. **Meta Title**: должен быть уникальным для каждой категории и содержать ключевые слова
2. **Meta Description**: краткое описание содержимого категории, привлекающее внимание
3. **Meta Keywords**: 5-10 релевантных ключевых слов через запятую

## Примеры

### Хорошие примеры

**Meta Title**: "Видео категории Спорт - Лучшие спортивные ролики"

**Meta Description**: "Смотрите лучшие спортивные видео онлайн. Футбол, баскетбол, теннис и другие виды спорта в HD качестве."

**Meta Keywords**: "спорт, видео, футбол, баскетбол, спортивные ролики, онлайн"

### Плохие примеры

**Meta Title**: "Спорт" (слишком короткий)

**Meta Description**: "Видео" (не информативно)

**Meta Keywords**: "видео, видео, видео, смотреть, смотреть, смотреть" (повторы и нерелевантность)
