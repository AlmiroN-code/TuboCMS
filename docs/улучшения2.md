### ✅ Улучшенный алгоритм связанных видео
- Обновлён `src/Service/RecommendationService.php`
- Новый алгоритм ранжирования по релевантности:
  1. **Серии** (100 баллов) — видео из той же серии
  2. **Исполнители/модели** (80-95 баллов) — видео с теми же моделями
  3. **Теги** (50-75 баллов) — видео с общими тегами
  4. **Категории** (30-45 баллов) — видео из тех же категорий
  5. **Автор** (20 баллов) — видео того же автора
  6. **Популярные** (10 баллов) — fallback на популярные видео
- Учитывается количество общих элементов (больше общих тегов = выше релевантность)
- При равной релевантности сортировка по просмотрам
- Обновлён `src/Controller/VideoController.php` для использования `RecommendationService`

Эти улучшения значительно повысят видимость сайта в поисковых системах и улучшат пользовательский опыт.
